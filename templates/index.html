About

Deceptive articles have been there since the advent of the internet. The increasing popularity of social media has also increased the number of fake articles. Fake news can be definedas fictitious articles deliberately fabricated to deceive readers. Social media and all type ofnews outlet publish deceptive articles to increase their readership.

The main goal of deceptive article websites is to affect public opinion on certain matters. Deceptive article detection is an important yet challenging problem that is not yet understood well.

Fake articles have mainly three characteristics, they are the text of an article, the user response it receives and the source users promoting it. There are many methods to detect fake articles. The hybrid ensembled classification model is built to increase the accuracy of detecting fake news.

The algorithms used in the hybrid classification model are Naive Bayes, Random forest, K nearest neighbor,Support Vector Machine, and Logistic regression. All these algorithms use training data as a bag of words model which was created using count vectorizer and term frequency-inverse document frequency transformer. Python is used as a language to verify and validate the result. Implementation is carried out using default algorithm values. The main aim is to find a reliable and accurate model that classifies a given article as either fake or real.
